
# 모음퀴즈 (moeum-quiz)

모음만 공개된 문장에서 **자음을 추리**하며 숨겨진 문장을 완성해 나가는 한글 기반 퍼즐 게임입니다.  

---

## 🎮 게임 방법

1. **모음은 모두 공개**된 상태에서 게임이 시작됩니다.  
2. 화면 아래 **자음 버튼**을 클릭해 해당 자음이 포함되어 있는지 추리합니다.  
3. 정답 자음은 해당 위치에 표시되고, 오답 자음은 **시도 횟수**가 차감됩니다.  
4. 3번의 기회 안에 문장을 완성하면 **승리**!  
5. 실패하면 **정답 문장**이 공개됩니다.

---

## ✨ 주요 특징

- **한글 중성 구조**에 착안한 창의적 게임
- 속담, 명언, 따뜻한 말, 영화·책 제목 등 **다양한 워드풀**
- 힌트 제공으로 **감성 추리력** 발휘
- **HTML·JS** 단일 파일, 어디서나 플레이
