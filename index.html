넘즐X

🎮 게임 소개

넘즐 X (NUMZZLE X)=넘버(NUMBER) + 퍼즐(PUZZLE)+X

넘즐은 수학 감각과 창의력을 발휘해야 하는 재미있는 계산 게임입니다.
한 라운드마다 주어진 카드로 최대값이 되는 수식을 만들고, 10라운드가 끝난 후 더 큰 숫자를 을 만든 사람이 점수를 얻습니다.

📋 게임 규칙

  • npc와 사용자는 서로 다른 숫자카드와 연산카드를 받습니다.
  • 주어진 카드로 수식을 만들어 최대값이 되는 수식을 제출합니다.
  • 실수했을 경우 초기화 버튼으로 수식을 다시 만들 수 있습니다.
  • 총 10라운드를 진행하며, 누적 점수가 더 높은 쪽이 승리합니다!

🎮 Game Overview

NUMZZLE X = NUMBER + PUZZLE +X

Numzzle X is a fun and strategic math game that challenges your sense of numbers and creativity.

📋 Game Rules

  • You and the computer gives different cards.
  • In each round, players create an expression that yields the highest possible value using the given cards.
  • If you make a mistake, use the reset button to start your expression over.
  • After 10 rounds, the player who has created the greater total score wins.
