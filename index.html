넘즐X

숫자카드와 연산기호 카드로 수식을 만드세요.

넘즐 X (NUMZZLE X)=넘버(NUMBER) + 퍼즐(PUZZLE)+X

넘즐은 수학 감각과 창의력을 발휘해야 하는 재미있는 계산 게임입니다.
한 라운드마다 주어진 카드로 최대값이 되는 수식을 만들고, 10라운드가 끝난 후 더 큰 숫자를 을 만든 사람이 점수를 얻습니다.

📋 게임 규칙

  • npc와 사용자는 서로 다른 숫자카드와 연산카드를 받습니다.
  • npc 와 사용자는 주어진 카드를 모두 사용해야 합니다.
  • 주어진 카드로 수식을 만들어 최대값을 만드는 게임은 넘즐초 X맥시, ,넘즐잼 X맥시, 넘즐X 맥시가 있습니다.
  • 주어진 카드로 수식을 만들어 최소값을 만드는 게임은 넘즐초 X미니, ,넘즐잼 X미니, 넘즐X 미니가 있습니다.
  • 실수했을 경우 초기화 버튼으로 수식을 다시 만들 수 있습니다.
  • 수식의 결과는 각 라운드의 점수가 됩니다.
  • 총 10라운드를 진행하며, 멕시 게임은 누적 점수가 더 높은 쪽이 승리합니다!
  • 총 10라운드를 진행하며, 미니 게임은 누적 점수가 더 낮은 쪽이 승리합니다!

Create expressions with given cards.

NUMZZLE X = NUMBER + PUZZLE +X

Numzzle X is a fun and strategic math game that challenges your sense of numbers and creativity.

📋 Game Rules

  • NPC and user receive different number cards and operation cards.
  • Users and computers must use all given cards.
  • Games where you create expressions with given cards to make the maximum value are Number Cho X Maxi, Number Jam X Maxi, and Number X Maxi.
  • Games where you create expressions with given cards to make the minimum value are Number Cho X Mini, Number Jam X Mini, and Number X Mini.
  • If you make a mistake, you can use the reset button to recreate the expression.
  • The result of the expression becomes the score for each round.
  • A total of 10 rounds are played, and in Maxi games, the side with the higher cumulative score wins!
  • A total of 10 rounds are played, and in Mini games, the side with the lower cumulative score wins!
